# 🚀 Fraud Detection ML API

A real-time Machine Learning API for detecting fraudulent transactions, deployed using Render.

---

## 📌 Overview

This project exposes a trained ML model as a REST API that can:

- Predict fraudulent transactions  
- Handle real-time requests  
- Be accessed from anywhere via HTTP  

---

## 🛠️ Tech Stack

- Python  
- Flask  
- Scikit-learn  
- Gunicorn  
- Render
